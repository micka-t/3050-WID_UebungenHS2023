2.2)    Für die grundlegenden Style wurde ein Klassendefinition mit CSS gewählt und für die einzelnen
        Hintergrundfarben jeweils im Inline einen Style definiert.

2.3)    Das span Element wird egal wie die grösse definiert wurde im Style CSS nur so gross dargestellt
        wie der Text ist. Wenn kein Text vorhanden ist wird nur ein dünner Strich dargestellt.

2.5)    Positionierung: "center" = Zentriert aneinandergereiht
                        "left" = Linksbündig aneinandergereiht
                        "right" = Rechtsbündig aneinandergereiht
                        "space-between" = Zentriert mit Abstand zwischen Elementen
                        "space-around" = Zentriert mit Abstand zwischen Elementen und ausserhalb der Elemente
